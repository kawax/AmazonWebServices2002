Keyword Searches

AWS 3.0 features a new parameter that you can use to generate search results based on a keyword. By adding this parameter to a search request--say, a search for new DVDs--you can narrow the results to fit a keyword of your choosing. This search will work with all searches except power search, similarities search, listmania search, wishlist search, and seller searches.

Example

To filter by keyword, add the keyword parameter to your request as shown:

http://xml.amazon.com/onca/xml3?t=webservices-20&dev-t=[developer’s token goes here]&BrowseNodeSearch=1000&mode=books&type=heavy&page=1&keywords=dog&f=xml

Sort Types

This section shows the possible ways to sort data for the various types of searches Amazon Web Services offers. 

Amazon.com Web Services search results can be sorted in various ways, depending on the type of product returned by the search.  Search results may be sorted by the following criteria:

  • Featured Items
  • Bestselling
  • Alphabetical (A-Z and Z-A)
  • Price (High to Low and Low to High)
  • Publication or Release Date
  • Manufacturer
  • Average Customer Review
  • Artist Name

Please note:  If a sort parameter isn’t specified in the request, the results will be ordered as if the +salesrank value is specified.

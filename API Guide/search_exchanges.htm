Exchange Searches

Exchange Searches via XML

Every item in the Amazon.com catalog that is offered by a third-party seller has a unique “exchange” ID.

To find a product’s Exchange ID, visit the page within our zShops area of the Web site that is devoted to that product. The ID appears in that page’s URL after the “/ts/exchange-glance” string. For example, in the following URL, the exchange ID would be Y02Y4715852Y4685018:

http://s1.amazon.com/exec/varzea/ts/exchange-glance/Y02Y4715852Y4685018

Call and Response

Request Format

http://xml.amazon.com/onca/xml3?t=webservices-20&dev-t=[Developer’s token goes here]&ExchangeSearch=[Exchange ID goes here]&type=[lite or heavy]&f=xml

Response

The ExchangeSearch request returns a ListingProductInfo node. The ListingProductInfo node contains an array of ListingProductDetails nodes.

Example

http://xml.amazon.com/onca/xml3?t=webservices-20&dev-t=[developer’s token goes here]&ExchangeSearch=Z01Y1142250Y1355111&type=lite&f=xml

Exchange Searches via SOAP

This section shows how to create a request for exchange searches.  This request will return a SOAP object containing the Lite or the Heavy form of the data.

Call and Response

Request Name

ExchangeSearchRequest

Request Format

exchange_id  string

mode  string

tag  string

type  string

devtag  string

Response

The ExchangeSearch request returns a ListingProductInfo node. The ListingProductInfo node contains an array of ListingProductDetails nodes.

Using the Remote Shopping Cart

This section explains how to use the remote shopping cart to add items to a cart and manage that shopping cart from a remote Web site or application.  This enables you to keep the customer on your site or application for the duration of their shopping session.

Elements of Remote Shopping Cart Calls

The following elements are returned by the Add call. The Cart ID and HMAC values returned by that call serve to identify the shopping cart and must be passed unchanged to the other shopping carts requests described in this section.

  1. Cart ID.  The Cart ID is the unique identifier for a given customer.
  2. HMAC.  The HMAC is a security token that must be passed back to Amazon Web Services for using an existing cart. This token is uniquely associated with each cart. When using the REST API, HMAC must be URL-encoded, so the + sign in the URL must be converted to %2B (otherwise the browser will interpret this as a space).
  3. Purchase URL.  The purchase URL must be included in your link to enable purchasing from a cart.  The purchase URL transfers items from your local cart to the Amazon.com shopping cart.  Your site or application must provide a user interface which allows the user to "click" on this URL and to display the result.

Remote Shopping Cart Process

This section shows the process for creating a cart, modifying the cart, and uploading the cart to Amazon for processing.

Creating and managing a remote shopping cart involves one or more of the following steps:

  1. Creating the Cart.  In order to create the cart, you simply need to send an add item request to AWS that contains an item for inclusion in the cart. 
  2. Adding Items.  As your customer shops, they may decide to add items to their cart.  You can enable this with the Add Items request.
  3. Modifying the Cart.  Your customer may decide to delete an item or change the number of a given item.  You enable this with the Modify Cart request.  You can also retrieve items from the cart or clear the cart entirely.
  4. Transfer the cart to Amazon.  When the customer is ready to make the purchase, you upload the cart to Amazon for order processing using the Purchase URL.  Please note:  Once the customer clicks on the purchasing URL, remote interaction with the cart is no longer possible.  Any modifications to the cart at this point must be made on Amazon.com

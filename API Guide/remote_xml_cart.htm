Using the Remote Shopping Cart via XML/HTTP

Creating a Cart and Adding an Item

To create a new shopping cart on a remote site using XML/HTTP, make the request shown in the table below.

Request Format

http://xml.amazon.com/onca/xml3?ShoppingCart=add&f=xml&dev-t=[developer’s token goes here]&t=[associates ID goes here]&Asin.[ASIN goes here]=[quantity goes here]

Function

The Cart Creation request creates a shopping cart and adds an item to that shopping cart.

Please note: the "&Asin.[ASIN goes here]=[quantity goes here]" portion of the URL can be repeated to add multiple items to the cart.

Adding Items to an Existing Cart

To add items to an existing cart on a remote site, make the request shown in the table below.  In order to add items to an existing cart, you’ll need to know both the CartID and the HMAC for that cart.

Request Format

http://xml.amazon.com/onca/xml3?ShoppingCart=add&f=xml& dev-t=[ [developer’s token goes here]&t=[associates ID goes here]&Asin.[ASIN goes here]=[quantity goes here] &CartId=[cart ID goes here] &Hmac=[HMAC goes here]

Function

The Add Item call adds an item to an existing shopping cart.

 

Modifying an Existing Cart

To modify an item that is already in the shopping cart, make the request shown in the table below.  In order to modify an existing cart, you’ll need to know the CartID, the HMAC, and the item number for the item you want to modify.  The item number is the itemID returned from any cart operation.  Note that the ShoppingCart= command now reads ShoppingCart=modify.

Request Format

http://xml.amazon.com/onca/xml3?ShoppingCart=modify&f=xml& dev-t=[ [developer’s token goes here]&t=[associates ID goes here]&Item.[itemID goes here]=[quantity goes here] &CartId=[cart ID goes here] &Hmac=[HMAC goes here]

Function

The Modify Cart call modifies items in an existing shopping cart.

Please note:  Once an item is placed in a cart, the user can change the quantity for that item to any number greater than zero. 

Retrieving Items from a Cart

To retrieve items from a cart on a remote site, make the request shown in the table below.  Note that the ShoppingCart= command now reads ShoppingCart=get.

Request Format

http://xml.amazon.com/onca/xml3?ShoppingCart=get&f=xml& dev-t=[ [developer’s token goes here]&t=[associates ID goes here]&CartId=[cart ID goes here] &Hmac=[HMAC goes here]

Function

The Retrieve Items call retrieves items in an existing shopping cart.

Clearing a Cart

To clear a new shopping cart on a remote site, make the request shown in the table below.  Note that the ShoppingCart= command now reads ShoppingCart=clear.

Request Format

http://xml.amazon.com/onca/xml3?ShoppingCart=clear&f=xml& dev-t=[ [developer’s token goes here]&t=[associates ID goes here]&CartId=[cart ID goes here] &Hmac=[HMAC goes here]

Response

The Clear Cart call clears the contents of an existing cart.

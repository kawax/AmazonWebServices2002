Using Amazon.com Chat

Amazon.com Chat is a great way to add value to your Web site, and to earn referral fees for doing so. With Amazon.com Chat, you can interact with your site visitors, and allow your site visitors to interact with each other. Creating a community and holding meetings can be as easy as choosing a product category, scheduling a time to meet, and promoting the service on your website!

Amazon.com Chat provides a forum for customers to communicate with each other in real-time and get up-to-the-minute product recommendations based upon the topics and items being discussed in the room. Since you will insert your Associates ID into your code before placing it on your site, you earn referral fees when chat members click through to purchase on Amazon.com

Below is the code for each of the chat rooms that we offer. Be sure to insert the correct Associate ID, Browse Category ID, and Browse Category Name into the code before adding it to your site.

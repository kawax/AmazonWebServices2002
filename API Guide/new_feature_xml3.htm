Using XML3

All existing calls to Amazon servers should now point to the AWS 3.0 services. Amazon will continue to support XML2 services, but you'll need to convert your calls to take advantage of new features in our 3.0 release.

To point to the new servers, simply change the XML2 element of your call to XML3.

Old Format

http://xml.amazon.com/onca/xml2?t=webservices-20...

New Format

http://xml.amazon.com/onca/xml3?t=webservices-20...

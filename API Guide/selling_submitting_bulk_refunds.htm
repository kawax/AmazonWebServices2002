Types of Requests (Detailed)

Submitting Bulk Refunds

Use Case: A seller should use the Bulk Refund request when they need to issue a large number of buyer refunds at one time.

HTTPS POST request to the following URL:

https://secure.amazon.com/exec/panama/seller-admin/catalog-upload/batch-refund

Header Content:

  • Username (using basic authentication)
  • Password (using basic authentication)
  • Developer Token
  • Order ID
  • Payments Transaction ID
  • Refund Amount
  • Reason
  • Message

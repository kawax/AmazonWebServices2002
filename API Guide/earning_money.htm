Earning Money through the Amazon Associates Program

If you find yourself sending a lot of traffic to Amazon.com through sites and applications that you have developed through Web Services, you may decide that you want to be compensated for that traffic. Fortunately, Amazon.com already has an existing program - the Associates Program -- that enables Web site owners to link to our store and earn referral fees for any sales that are generated through their links.

Embedding Your Associates ID in Your Request

Each Associate is assigned an Associates ID, and this same ID can very easily be embedded into any XML/HTTP or SOAP calls made to the Amazon.com Web site.

To do so, simply replace the "webservices-20" tag found in the examples with your Associates ID. This will have the effect of automatically inserting your Associates ID in product links within the XML document. In fact, if you already have an application that parses the XML data and uses product links contained within that data, you won't have to change your application in any way - simply replace the Associates ID and you are all set. (The same is true if you are using our XSLT service to generate pre-formatted HTML widgets).

Using Your Associates ID with International Web Services

If you intend to make calls to multiple versions of Amazon Web Services (US, UK, Germany and/or Japan), you’ll need to use different Associate IDs in each of your calls.  

For example, say a developer has a developer token of "GTYDRES564THU". They can use this same token for any call that they make, regardless of which country's services they are calling. But if they are also an Associate for all four countries, then they will have four separate Associate IDs (say -- myWebsite-20 (US), myWebsite-21 (UK), myWebsite-22 (DE), and myWebsite-23 (JP)), and they will need to use the correct ID in their calls (depending on which country they are calling).

The four example calls might look like this: 

  • US:  http://xml.amazon.com/onca/xml3?t=myWebsite-20&dev-t=GTYDRES564THU&KeywordSearch=dogs&mode=books&type=lite&page=1&f=xml
  • UK:  http://xml-eu.amazon.com/onca/xml3?t=myWebsite-21&dev-t=GTYDRES564THU&KeywordSearch=dogs&mode=books-uk&type=lite&page=1&f=xml&locale=uk
  • DE:  http://xml-eu.amazon.com/onca/xml3?t=myWebsite-22&dev-t=GTYDRES564THU&KeywordSearch=dogs&mode=books-de&type=lite&page=1&f=xml&locale=de
  • JP:  http://xml.amazon.com/onca/xml3?t=myWebsite-23&dev-t=GTYDRES564THU&KeywordSearch=dogs&mode=books-jp&type=lite&page=1&f=xml&locale=jp

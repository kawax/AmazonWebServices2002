This software development kit provides information on using Amazon Web Services to add content to Web sites, enable transactions, and upload products to Amazon.com. The guide is broken into the following areas:

  • Getting Started
  • New in this Release
  • Product and Catalog Data
  • Using International Data
  • Enabling Transactions
  • Selling Products on Amazon.com
  • Reference

The guide also provides some general information about using Web services, as well as pointers to additional reference sources and instructions on how to access developer message boards and other resources.

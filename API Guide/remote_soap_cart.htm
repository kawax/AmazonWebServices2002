Using the Remote Shopping Cart via SOAP

Creating a Cart with SOAP

To create a new shopping cart on a remote site using SOAP, make the request shown in the table below.

Request Name

AddShoppingCartItemsRequest

Request Format

tag string

devtag string

CartId string

HMAC string

Items array

Response

The AddShoppingCartItemsRequest request returns an object of type ShoppingCartResponse.

Response

Modifying an Existing Cart using SOAP

To modify an item that is already in the shopping cart, make the request shown in the table below.  In order to modify an existing cart, you’ll need to know the CartID, the HMAC, and the item number for the item you want to modify.  The item number is the itemID returned from any cart operation. 

Request Name

ModifyShoppingCartItemsRequest

Request Format

tag string

devtag string

CartId string

HMAC string

Items array

Response

Returns an object of type ShoppingCartResponse.

Please note:  Once an item is placed in a cart, the user can change the quantity for that item to any number greater than zero.

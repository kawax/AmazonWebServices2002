Marketplace Searches via XML

There are two new types of marketplace searches -- listing-ID and keyword. A listing-ID search allows you to get product information for a specific marketplace item. Every listing ID is unique to a marketplace item, so requesting that ID will give you the product information for that item. Every marketplace item has both a listing ID and exhcange ID, and prior to this release we had exchange ID searches, but many sellers didn't know their exchangeID.

The second type of new marketplace search is a keyword search. A keyword search enables you to search for across all of our marketplace listings for items that match your keyword criteria. This search also can use a number of optional parameters:

  • keyword-search: Can have the following two values:
    • title: search for the keyword in the title of exchanges only
    • titledesc: search for the keyword in the title & description exchanges only
  • browse-id: The marketplace browse-id’s, list is similar to the seller-browse-id
  • zipcode: To filter marketplace searches by sellers from a specific part of the country, fill in the appropriate ZIP code with your request
  • area-id: The numerical country code, it is the list of numbers that can be picked up by looking at the source code in the following URL: http://s1.amazon.com/exec/varzea/subst/search/fixed-search.html/
  • geo: values are:
    • ship-to: Indicates that the items can be shipped to the area-id picked above.
    • ship-from: Indicates that show items that are shipped from the area-id specified above
  • rank: The sort-type values are:
    • –bfp: sorted by featured listings (default)
    • –startdate: Start Date from New to Old
    • +startdate: Start Date from Old to New
    • +price: Price Low to High
    • –price: Price from High to Low
  • index: values
    • marketplace: return marketplace offerings only
    • zshops: return zshops listings only.

Call and Response: Listing ID

Request FormatWorking with z-Shops Products

http://xml.amazon.com/onca/xml3?t=webservices-20&dev-t=[Developer’s token goes here]&MarketplaceSearch=listing-id&t=test&f=xml&type=[lite or heavy]&listing-id=[listing ID goes here]

Response

The Marketplace search request returns data from Amazon's marketplace stores filtered for sellers.

Request Name

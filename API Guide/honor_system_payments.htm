Enabling Payments with the Amazon Honor System

Using the Amazon Honor System, you can accept payments directly on your Web site or application.

The Amazon Honor System enables you to collect payments from your users using Amazon's payment system. For example, you might ask for a donation from your users, or collect voluntary payments for Web site features.

Using a link, you can add the following payment box to your site:

To add the box to your page, complete the following steps.

  1. Go to http://s1.amazon.com/exec/varzea/subst/fx/home.html/104-8908261-1003914 and create your own pay-page ID. Now that you have a pay-page ID, you can add that ID to your link to enable payments.
  2. Add the following link to your Web site or application: http://s1.amazon.com/exec/varzea/dt/assoc/tg/aa/assoc/assoc/-/[Honor_System_pay_page_id]
  3. When customers click the link, your pay-box will open in a pop-up window. When a customer enters an amount and clicks the Pay now! button, you will receive a payment from Amazon.

For more information about Amazon's Honor System program, please see the Honor System page.

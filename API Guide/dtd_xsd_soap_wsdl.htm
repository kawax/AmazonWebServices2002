Lite Request Results

Product Info Node

This section provides details on the product info node for lite requests. 

Element Name

Description of Element

TotalResults

Number of results (products) generated by search.

ListName

Title of Listmania list.

Details

Details about a specific product. See Details node below.

ErrorMsg

The error message that is returned if the call fails.

Details Node

This section provides details on the details node for lite requests. 

Element Name

Description of Element

URL

Link to product detail page at Amazon.com (includes an embedded Associates ID).

Asin

Amazon.com Standard Item Number.

ProductName

Name or title of product.

Catalog

Product category (book, music, etc.).

Artists

Musician(s) of a CD or cassette.

Authors

Author(s) of a book.

ReleaseDate

Date that the product was released to market.

Manufacturer

Manufacturer of product. Can represent publishers (for books), labels (for music) or studios (for films).

ImageUrlSmall

Small product image or cover art.

ImageUrlMedium

Medium product image or cover art.

ImageUrlLarge

Large product image or cover art.

ListPrice

Usually represents the price of the product suggested by the manufacturer.

OurPrice

The Amazon.co.uk product price.

Status

Indicates if we encountered any errors while processing the service request.

Working with Offerings

An offering is an actual item that Amazon or a third party merchant has for sale.  When offer=used (or any of the other values), Used, ThirdPartyNew, Collectible, and Refurbished data will appear accordingly if the type is set to heavy, regardless of the type of call.  This is true for any call that returns product data, including the following:

  • Keyword
  • Actor
  • BrowseNode
  • Director
  • Artist
  • Author
  • Power
  • Manufacturer
